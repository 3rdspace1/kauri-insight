## üìä Data Visualization Visual Language

Different data types require different visual approaches:

### Quantitative Data

**Visual Forms**:
- Bar lengths, circle sizes, area scales, linear progressions
- Mountains of data, rivers of information, forests of variables
- Grid-aligned, scale-conscious, proportion-accurate

**Best For**: Scores, ratings, measurements, quantities

### Qualitative Data

**Visual Forms**:
- Color categories, shape variations, texture distinctions
- Color spectrums, pattern diversity, categorical landscapes
- Distinct groupings, clear separations, categorical clarity

**Best For**: Themes, categories, open-ended responses

### Temporal Data

**Visual Forms**:
- Flowing timelines, progressive sequences, rhythmic patterns
- Time rivers, progression paths, evolution flows
- Sequential accuracy, temporal rhythm, chronological flow

**Best For**: Trends over time, before/after, historical data

### Categorical Data

**Visual Forms**:
- Grouped clusters, segmented regions, classified zones
- Organized territories, classified landscapes, sorted domains
- Clear boundaries, distinct categories, organized spaces

**Best For**: Demographics, segments, classifications

## üö´ Negative Prompt Engineering

Comprehensive avoidance system prevents common AI pitfalls:

### Text Artifacts
**Avoid**: text, words, letters, numbers, labels, typography
**Reason**: AI models struggle with accurate text rendering

### Human Elements
**Avoid**: people, faces, hands, human figures, portraits
**Reason**: Unnecessary for data visualization, can distract

### Logo Confusion
**Avoid**: logos, trademarks, watermarks, signatures
**Reason**: Handled separately through proper channels

### Quality Issues
**Avoid**: blurry, pixelated, compression, grainy, noisy, distorted
**Reason**: Undermines professional presentation

### Inappropriate Styles
**Avoid**: cartoon, anime, sketch, amateur, unprofessional, childish
**Reason**: Not suitable for business intelligence

### Composition Problems
**Avoid**: cluttered, chaotic, messy, disorganized, confusing
**Reason**: Violates data visualization principles

### Industry-Specific Avoidances
- Healthcare: disturbing, medical gore, disease imagery
